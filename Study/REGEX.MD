## 1. 🔠 문자 클래스 (어떤 문자를 찾을까?)
특정 종류의 문자를 지칭하는 단축키입니다. 대문자는 소문자의 **반대(Not)** 역할을 합니다.

| 기호 | 의미 | 풀어쓴 형태 | 설명 |
| :--- | :--- | :--- | :--- |
| `\d` | 숫자 (Digit) | `[0-9]` | 0부터 9까지의 숫자만 찾음 |
| `\D` | 숫자가 아닌 것 | `[^0-9]` | 숫자를 제외한 모든 문자(알파벳, 기호, 공백 등) |
| `\w` | 단어 (Word) | `[a-zA-Z0-9_]` | 알파벳 대소문자, 숫자, 언더바(`_`) |
| `\W` | 단어가 아닌 것 | `[^a-zA-Z0-9_]`| 특수문자, 공백 등 |
| `\s` | 공백 (Space) | `[ \t\n]` | 스페이스바(띄어쓰기), 탭, 줄바꿈 |
| `.` | 모든 문자 | - | 줄바꿈을 제외한 글자 딱 1개 |

<br>

## 2. 🔢 수량자 (몇 개나 연속으로 나올까?)
앞의 문자나 그룹이 몇 번 반복되는지 지정합니다.

| 기호 | 의미 | 예시 | 설명 |
| :--- | :--- | :--- | :--- |
| `*` | 0개 이상 | `A*` | 'A'가 없거나 여러 개 통과 |
| `+` | 1개 이상 | `A+` | 'A'가 **최소 1개** 필수 |
| `?` | 0개 또는 1개 | `A?` | 'A'가 있거나 없거나 (선택적) |
| `{n}` | 정확히 n개 | `\d{4}` | 숫자 딱 4자리 (예: 1234) |
| `{n,m}`| n개 이상, m개 이하| `[a-z]{2,5}` | 소문자 알파벳 2~5글자 사이 |

<br>

## 3. ⚓ 앵커 (시작과 끝)
문자열 중간에 숨어있는 패턴이 아닌, **전체가 완벽히 일치하는지 검사**할 때 필수적으로 사용합니다.

* **`^`** : 문자열의 **시작**을 의미
* **`$`** : 문자열의 **끝**을 의미

<br>

## 4. 💡 실전 조합 패턴 (Snippets)
자주 쓰이는 패턴 모음입니다. 필요할 때 복사해서 사용하세요.

### 📱 휴대폰 번호
> `^010-\d{4}-\d{4}$`
* `^010-` : 무조건 '010-'으로 시작
* `\d{4}` : 숫자 4자리
* `-` : 하이픈
* `\d{4}$` : 다시 숫자 4자리로 끝남

### 🧑 한글 이름 (2~4글자)
> `^[가-힣]{2,4}$`
* `[가-힣]` : 한글(가~힣)로만 이루어짐
* `{2,4}` : 2글자에서 4글자 사이 길이 제한

### 📧 이메일 주소 (기본형)
> `^\w+@\w+\.\w+$`
* `\w+` : 아이디 (단어 1개 이상)
* `@` : 골뱅이 기호
* `\w+` : 도메인 이름
* `\.` : 점(.) 특수기호 처리 (이스케이프)
* `\w+$` : 최상위 도메인 (com, net 등)으로 끝남
